Javascript OBject Cheat SHeet

August 7th, 2016

  1. How to create a new Javascript Object
    • object literal: var name = {}; **(preferred because it's faster)
    • object constructor: var new = new Object();
  2. Add property to new object
    • name[firstName] = "Katie" - using dot notation
    • name.firstName = "Katie" -using dot notation
    • summarize key difference between using dots vs brackets
    • When using dot notation, the property is automatically invoked.
    • When using bracket notation, the property is evaluated first before it is called
    • We can also add a function name as the key of an object, and have the body of the function be the value
    • We can also add a function to an object
    • EX:
      var name = {
      sum: function(x,y) {
      return x + y;
      }
      name.sum(3+4);
      would return 7;
      }
  3. Changing the object
    • name[firstName] = "Brian"
    • name.firstName = "Brian"
    • delete name.firstName - this will delete the firstName from the name object
