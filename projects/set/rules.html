RULES OF THE GAME

How to Play

  • Grab as many sets as you can
  • You have one minute

    • What is a set?

  • A set is three cards that are either all the same or all different in each individual feature
  • There are a total of four features - color, shape, quantity, and pattern
  • No two cards are the same
  • This is a set. Notice that all 3 cards have either every feature in common (quantity and pattern), or everything different(shape and color)
  • This is also a set. The color and pattern is all the same, and the quantity and shape are different
